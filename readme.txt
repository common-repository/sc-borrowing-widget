SmartCredit.io Loan Calculator Widget allows calculation of the loan amounts and the loan collateral amounts.
 
== Description ==
 
The Borrowing Widget allows the user to borrow stablecoin or Ethereum directly via their Metamask wallet or WalletConnect without intermediaries.

You are your own bank!

At the same time, the integrating website will earn the loan origination fees and reward tokens for the referred users.
 
== Installation ==
 
This section describes how to install the plugin and get it working.

1. Upload `SC-BorrowingWidget-plugin.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the shortcode `[borrowing-widget referral="" width="" height=""]` in your templates
